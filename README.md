# Finance Executive Dashboard - AICTE Microsoft Elevate Internship

## 📌 Project Overview:

This repository contains a Finance Executive Dashboard developed as part of the AICTE Microsoft Elevate Internship Program. The project demonstrates the transformation of raw financial data into an interactive visual report that provides key business insights into global sales, profitability, and market trends.

## 📊 Key Insights & Analytics:

Based on the analysis of the global financial dataset (2013-2014):

**Total Revenue:** Managed and visualized $103.34M in gross sales.

**Profitability:** Tracked $14.78M in total profit, identifying the United States as the top-performing country.

**Product Performance:** Identified "Paseo" as the primary revenue driver across multiple segments.

**Market Segments:** Analyzed performance across Government, Midmarket, Enterprise, and Small Business sectors, with the Government segment showing the highest volume.

## 🛠️ Tools & Technologies:

**Power BI:** Data Modeling, DAX (Data Analysis Expressions), and Visualization.

**Microsoft Excel:** Data cleaning and initial preprocessing.

**Data Transformation:** Power Query used for ETL (Extract, Transform, Load) processes.

## 📂 Files in this Repository:

**Finance Report.pbix:** The complete Power BI Dashboard file.

**Financial Sample.xlsx:** The raw dataset used for the analysis.

## 🚀 How to Use:

1. Clone or download this repository

2. Open Finance Report.pbix in Power BI Desktop

3. If prompted, reconnect the data source to Financial Sample.xlsx

4. Explore the interactive dashboard
